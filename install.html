How to install...

Requirements:

  • Minecraft 1.7.10 (no you can't use 1.8 or any previous version) - link
  • Minecraft Forge 1.7.10 (latest) - link
  • Arduino IDE (drivers, libs, etc.)
  • On Mac OS and Linux you can also install ino tool

Installation:

We expect that you have Minecraft and Minecraft Forge installed, so this won't be gathered in this tutorial.

preparing Arduino to communication with Minecraft

At first, you have to upload our Arduino sketch to your Arduino (UNO or Mega2560). It is very easy, here are the steps:

Using Arduino IDE (any OS)

  1. Open Arduino IDE
  2. Click File > Open, then go to folder where you extracted downloaded zip
  3. In this folder, go to arduino_sources/src/ where you find "mineduino.ino", thats the file you have to open
  4. Select your Board and Serial port in Tools menu
  5. Finally, you can click the right facing arrow (Upload)

Using command line ino tool (Linux, OS X)

We are preparing this tutorial...

Mineduino mod installation

So you have your Arduino ready and connected to computer? Now you have to install our Forge mod. The latest version is already packed in the zip you downloaded.

  1. Copy mod .jar file to .minecraft/mods
  2. Start Minecraft (do not open any world) and close it (our mod configuration file is generated)
  3. Edit the configuration file, it's in .minecraft/config/mineduino.cfg
  4. You can now start Minecraft and control your Arduino with it!!!

If you want to disconnect your Arduino, you have to first leave the world you are in, same before opening a world you must have your Arduino connected to your computer and setup correct Serial port in mod settings
